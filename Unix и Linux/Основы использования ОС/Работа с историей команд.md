Основная команда, которой можно воспользоваться в терминале -- `history`. Удобнее использовать с пейджером, например `less`.

Совместно с `history` можно использовать символ `!` . Его возможности:
* `!<номер_команды>` -- выполнение команды с указанным номером (из `history`)
* `!<текст>` -- выполнение последней команды, начинавшейся с указанного текста
* `!<номер или текст>:<номер>` -- берет определенное слово из указанной команды
* `!!` -- последняя введенная команда
* `!:<номер>` -- частный случай, определенное слово из последней команды

Также можно использовать поиск по подстроке в истории команд. Выполняется сочетанием клавиш `Ctrl-R`, затем вводом желаемой команды. Если нажать `Ctrl-R` снова, то получим *следующую* (более старую) команду, содержащую *ту же подстроку*.
