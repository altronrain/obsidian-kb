При описании многострочных макросов (директива `%macro`) можно задать переменное число параметров. Это делается с помощью символа тире
Примеры:
```nasm
%macro mymacro 1-3           ; принимает от 1 до 3 параметров
%macro mysecondmacro 2-*     ; принимает не менее 2 параметров до бесконечности
```

При этом индексирование предусмотрено только до 9 передаваемого параметра (`%9`)
(`%0` -- хранит число параметров)
Для обращения к произвольному параметру можно использовать директиву `%rotate`
Она принимает *один* аргумент, указывающий на сколько позиций сдвинуть номера параметров (вправо).
Так, после указания:
```nasm
%rotate 1
```
Параметр, имевший номер `%2`, станет доступен через `%1`
При этом с тоит запомнить что в первый (предыдущий элемент) попадет общее количество параметров `%0`, которое в ротации как бы не усчаствует
Директива может принимать отрицательные значения в качестве аргументов, при этом сдвиг будет выполнен влево:
```nasm
%rotate -1
```

Пример вариации многострочного макроса для вызова процедуры `call` с произвольным числом параметров:

Так при вызове:
```nasm
pcall myproc, eax, myvar, 27
```
Будет преобразован в:
```nasm
push dword 27
push dword myvar
push dword eax
call myproc
add esp, 12
```
Напоминание, что параметры заносятся в стек в обратном порядке, таким образом наверху останется первый параметр
