*Условная компиляция* (включение определенных фрагментов кода в компилируемую программу *по условию*) в NASM выполняется средствами макропроцессора.

Для условной компиляции используются директивы:
* `%define`
* `%ifdef`
* `%endif`
Код, помещенный в блок директив `%ifdef <однострочный_макрос>`, `%endif` будет выполнен, только если этот однострочный макрос определен директивой `%define <однострочный_макрос>`
Пример:
```nasm
%define DEBUG_PRINT
;...
%ifdef DEBUG_PRINT
	PRINT "Entering section"
%endif
;
; исследуемая секция
;
%ifdef DEBUG_PRINT
	PRINT "Leaving section"
%endif
```
Код, печатающий текст, будет выполнен. Если же закомментировать `%define`, то код не выполнится

Можно вообще не добавлять в код `%define`, а воспользоваться ключом `-d` при вызове компилятора:
```shell
nasm -f elf -dDEBUG_PRINT prog.asm
```
