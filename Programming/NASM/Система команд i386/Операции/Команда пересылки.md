Команда `mov` -- команда пересылки данных из одного места в другое. Данная комнада только копирует данные, не выполняя никаких преобразований.
```nasm
mov var1, var2
```
Имеет два операнда:
* первый задает назначение (место, куда будут помещены данные)
* второй задает источник (место, откуда будут взяты данные)

Первый операнд может быть регистровый или типа "память" (адресный).
Второй операнд может быть любого типа.
> [!NOTE] Мнемоники и машинные команды
> Важно помнить, что одной мнемоникой могут обозначаться различные машинные команды, выполняющие схожие действия (для удобства программиста)
> Так на примере `mov`:
> ```nasm
> mov eax, 2
> mov eax, [x]
> mov [x], eax
> mov [x], al
> ```
> указанные команды обозначают разные машинные команды, имеют разные значения машинного кода и даже занимают разное количество байт в памяти

Ассемблер (и само устройство процессора) не умеет работать с двумя операндами типа память. Такие операции нужно разбивать на несколько
```nasm
; Приведет к ошибке
mov [y], [x]

; Отработает штатно
mov eax, [x]
mov [y], eax
```
