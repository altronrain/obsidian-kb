NASM позволяет опираться на кодировочную таблицу ASCII, и чтобы не запоминать коды символов, указывать сами символы, заулюченные в апострофы или двойные кавычки

Пример выделения байта с кодом символа цифры 7:
```nasm
fig7    db '7'
```
Пример выделения однобайтового массива под фразу:
```nasm
welmsg  db 'Welcome to Cyberspace!'
```
строка из 22 символов будет располагаться по адресу `welmsg` в массиве однобайтовых ячеек, содержащих коды символов

Внутри апострофов двойные кавычки рассматриваются как обычный символ, и наоборот. Показательный пример на фразе: So I say: "Don't panic!"
```nasm
panic   db 'So I say: "Don', "'", 't panic!"'
```
