Наиболее распространенный компилятор на Unix-подобных ОС.

Всегда следует использовать по меньшей мере 2 флага при запуске `gcc`:
* `-Wall` -- включение *всех разумных* предупреждений
* `-g` -- генерация отладочной информации

Также крайне полезно указывать `-o` для указания имени результирующего файла (output). Без этого флага результат будет иметь имя `a.out`

Для некоторых файлов на Си приходится подключать библиотеки (во избежание ошибок на этапе компоновки). Делается это флагом `-l<имя_библиотеки>`. Данный флаг указывается *после* имен исходных файлов
