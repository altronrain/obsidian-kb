В языке C для доступа к параметрам, составляющим командную строку, можно получить описав функцию `main` *с параметрами*:
* первый элемент должен быть типа `int` -- он задает количество элементов командной строки
* второй -- адрес начала массива из указателей на слова, составляющие командную строку
	* каждый элемент этого массива имеет тип `char*`
	* соответственно адрес каждого элемента этого массива имеет тип `char**` (оно же `char *argv[]`)

Обычно эти аргументы функции `main` называются:
* `argc` -- *argument counter*
* `argv` -- *argument vector*
Пример заголовка функции `main` с параметрами:
```c
int main(int argc, char **argv)
```

В массиве `argv` элементов на `1` больше -- последний из них равен нулевому адресу `NULL`

Пример программы, выводящей на печать свои аргументы с циклом `for`:
```c
#include <stdio.h>
int main(int argc, char **argv)
{
	int i;
	for(i = 1; i < argc; i++)
		pritf("[%s]\n", argv[i]);
	return 0;
}
```
Пример той же программы через цикл `while` без использования `argc`:
```c
#include <stdio.h>
int main(int argc, char **argv)
{
	argv++;
	while(*argv) {
		printf("[%s]\n", argv[i]);
		argv++;
	}
	return 0;
}
```
