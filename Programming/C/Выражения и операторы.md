## Выражения

Выражение (*expression*) состоит из комбинации операций и операндов. В простейшем случае выражение -- отдельный операнд.

Примеры выражений:
```c
4
-6
4 + 21
a * (b + c / d) / 20
q = 5 * 2
x = ++q % 3
q > 3
```

Каждое выражение имеет *значение* (то есть может быть *вычислено*). Для того чтобы найти значение выражения необходимо выполнить *операции* над *операндами* в порядке *приоритета* (см. [[Приоритет операций]])

## Операторы

Оператор -- "строительный блок программы" (так как *программа* -- последовательность операторов).

В языке C оператор завершается символом `;` (который является *терминатором*, не сепаратором). Так:
```c
legs = 4     /* выражение */
legs = 4;    /* оператор */
;            /* пустой оператор -- простейший случай оператора */
```

Выражения, дополненные `;` можно назвать *операторами выражения*. Не все операторы являются операторами выражения (например, оператор объявления).

Операторы вида
```c
8;
3 + 4;
```
тоже считаются допустимыми, однако они не несут смысловой полезной нагрузки.

Обычно операторы изменяют значения переменных или вызывают функции, например:
```c
x = 25;
++x;
y = sqrt(x);
```

Оператор -- завершенная инструкция, однако не все завершенные инструкции являются операторами.
Например:
```c
x = 6 + (y = 5);
```
Выражение `y = 5` представляет собой завершенную инструкцию, но она является лишь частью оператора. По этому для идентификации инструкций, являющимися операторами используется `;`

### Виды операторов

Часто применяемые операторы приведены в следующем листинге:
```c
/* addemup.c -- виды операторов */
#include <stdio.h>

int main(void) {
	int count, sum; /* оператор объявления */

	count = 0; /* оператор присваивания */
	sum = 0; /* оператор присваивания */

	while (count++ < 20) { /* оператор цикла while */
		sum = sum + count;
		; /* пустой оператор */
	}

	printf("sum = %d\n", sum); /* оператор вызова функции */

	return 0; /* оператор возврата */
}
```

* *оператор объявления* -- устанавливает имена и типы переменных и выделяет для них пространство в памяти. Не является оператором выражения
* *оператор присваивания* -- присваивает переменной значение. Состоит из имени переменной, операции присваивания `=` и следущего за ней выражения. Является оператором выражения
* *оператор вызова функции* -- передает управление вызываемой функции, передает ей аргументы при наличии (выполняет необходимые операции над стеком в процессе вызова)
* *оператор цикла* while -- составной оператор (имеет 3 части):
	* ключевое слово `while`
	* условие проверки в круглых скобках
	* блок (составной оператор), который выполняется если условие истиннно
* *оператор возврата* -- завершает выполнение функции, передавая указанное значение в место вызова (также выполняет необходимые операции над стеком в процессе возврата из функции)
