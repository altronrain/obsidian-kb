В базовом комплекте набор встроенных типов языка C довольно мал. Фактически имеются:
* целые числа (различающиеся разрядностью и знаковостью)
* числа с плавающей точкой (различающиеся разрядностью)

Отсутствуют специальные типы для символов, логический тип, строковый тип.

Связано это с тем, что на языке ассемблера как раз процессор работает либо с числами с плавающей точкой, либо все остальное представляет в виде целых чисел.

Так, при разборе логического типа:
* `0` будет означать ложь
* иные целые числа будут означать истину
или же на адресных выражениях:
* нулевой указатель будет означать ложь
* любой другой указатель будет означать истину

Символы представляются для процессора в виде целых чисел (кодов символов). Вот и в C *символ и его код -- это одно и то же*.
