Характеристика объекта.

Характеризует постоянство объектов, доступных через идентификаторы (их время жизни/существования).

Объект в C имеет одну из следующих 4 продолжительностей хранения:
* статическую
* потоковую
* автоматическую
* выделенную

## Статическая продолжительность хранения
Статические объекты существуют н*а протяжении всего времени выполнения программы*.
Такими, например, являютстя *все* переменные с областью видимости в пределах файла.

> [!NOTE] Нюансы `static`
> Ключевое слово `static` для переменных с областью видимости в пределах файла
> отвечает за *связывание*
> 
> Ключевое слово `static` примененное внутри блока к локальной переменной создает ее со статической продолжительностью хранения (а не автоматической). Такая переменная будет расположена в статической памяти с момента загрузки программы в память (с момента запуска). Однако ее область видимости будет все еще ограничена блоком, в котором она находится.

## Потоковая продолжительность хранения
Важна в *параллельном* программировании, когда выполнение программы может быть разделено на несколько потоков.
Объект с потоковой продолжительностью хранения существует с момента его объявления и до завершения потока.

Такой объект создается, когда объявление объекта с областью видимости *в пределах файла* модифицировано с помощью ключевого слова `_Thread_local`.
Каждый поток получает свою собственную закрытую копию этой переменной.

## Автоматическая продолжительность хранения
Память, используемая для автоматических переменных, является *рабочим пространством (временной памятью)*, которая может применяться многократно.

Так. большинство переменных с областью видимости в пределах блока обычно имеют именно автоматическую область видимости. Память для них выделяется при входе в блок, и освобождается при выходе из блока. Таким образом данная память может быть далее задействована для следующей функции или иного блока.

> [!NOTE] Массивы переменной длины
> Являются небольшим исключением. Они существуют *от места объявления* до конца блока.

Большинтво локальных переменных попадают в категорию автоматической продолжительности хранения.
