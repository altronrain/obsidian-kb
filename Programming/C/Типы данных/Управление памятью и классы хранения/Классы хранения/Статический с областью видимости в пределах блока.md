Она же *локальная статическая переменная*

В словосочетании *статическая переменная*, которое кажется взаимисключающим, слово *статическая* отвечает за *область памяти*, а не за значение переменной.
Так, перменные с областью видимости в пределах файла автоматически являются статическими. Но можно создать и локальные переменные в пределах блока, имеющие статическую продолжительность хранения.

Параметры:
* Область видимости: В пределах блока
* Связывание: Отсутствует
* Продолжительность хранения: Статическая

Переменная в таком случае доступна только внутри блока (аналогично автоматическому классу хранения), однако *хранится в памяти все время выполнения программы*. Компилятор *запоминает значение* данной переменной от одного *вызова функции* до другого.

## Объявление таких переменных
Объявляются внутри блоков с применением ключевого слова `static`:
```c
int somefunc (void)
{
    static int counter = 0;
}
```

Так, демонстрационная программа:
```c
int main (void)
{
	for (int count = 1; count <= 3; count++)
	{
		printf ("Начинается итерация %d:\n", count);
		trystat ();
	}
	return 0;
}

void trystat (void)
{
	int fade = 1;
	static int stay = 1;
	printf ("fade = %d\tstay = %d\n", fade++, stay++);
}
```
Отлично иллюстрирует поведение данного класса хранения:
```
$ ./loc_stat
Начинается итерация 1:
fade = 1 stay = 1
Начинается итерация 2:
fade = 1 stay = 2
Начинается итерация 3:
fade = 1 stay = 3
```

## Инициализация таких переменных
Такие переменные инициализируются единожды при компиляции программы.

Такие переменные, если явно не проинициализированы каким-либо значением, инициализируются нулем `0`. (Потому что будет располагаться в секции `.data` а не `.bss`)

См. также [[Память пользовательской задачи. Сегменты]]

## Нюансы с размещением объявления и хранением
На самом деле подобные объявления типа:
```c
void trystat (void)
{
	static int stay = 1;
...
```
Не являются частью функции, внутри которой они расположены. При загрузке программы в память такие данные уже располагаются в секции в нужных местах.

Оператор помещается в функцию, чтобы сообщить компилятору, что только данной функции разрешено *видеть* данную переменную.
То есть данная конструкция не является оператором времени выполнения.

Модификатор `static` нельзя использовать для параметров функции.
