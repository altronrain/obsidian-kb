Исходя из названия класса такие переменные имеют:
* область видимости в пределах файла
* внутреннее связывание
* статическую продолжительность хранения

То есть это статическая переменная с областью видимости в виде *одной единицы трансялции*, не более.

## Объявление таких переменных
Объявляется такая переменная:
* за пределами всех функций с ключевым словом `static`
* внутри функции, в которой используется (с ключевым словом `extern`) (дополнительное необязательное объявление, аналогично переменным с внешним связыванием, так как оба *глобальные*)

Примеры:
```c
int traveler = 1;       /* статическая переменная с внешним связыванием */
static int stay = 1;    /* статическая переменная с внутренним связыванием */

int main (void)
{
	extern int traveler; /* использование глобальной переменной traveler */
	extern int stay;     /* использование глобальной переменной stay */
	...
}
```

Переменные `traveler` и `stay` обе являются глобальными. Но только переменную `traveler` можно применять в других единицах трансляции.
