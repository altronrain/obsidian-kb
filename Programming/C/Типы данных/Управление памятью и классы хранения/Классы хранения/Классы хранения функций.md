Помимо классов хранения переменных, существуют и классы хранения функций, для которых можно указать ключевые слова `extern` и `static`

Функция может быть:
* внешней (по умолчанию)
* статической
* встраиваемой (C99 стандарт)

Доступ к внешней функции могут получать функции в других файлах.
Статическая функция доступна только внутри файла, где она определена.

Пример:
```c
double gamma (double);          /* внешняя по умолчанию */
static double beta (int, int);  /* статическая функция */
extern double delta (int, int); /* внешняя функция */
```

Функции `gamma()` и `delta()` можно применять в других файлах (единицах трансляции), а функию `beta()` -- нельзя.

*Статические* функции помогают закрыть функцию в рамках конкретного модуля, единицы трансляции, что помогает *решить* потенциальную проблему *конфликта имен*.

На практике, ключевое слово `extern` часто явно прописывают при *объявлении* функции, *определенной в другом файле*. Так как в принципе все функции по умолчанию *extern*.
