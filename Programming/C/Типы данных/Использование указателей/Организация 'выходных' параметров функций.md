В C существует *только один* способ передачи параметров -- по значению

В ситуации, когда требуется передать переменную в функцию, приходится передавать *адрес переменной*, а в теле функции использовать разыменование для обращения к переменной

Пример функции, меняющей значение двух переменных типа `int`:
```c
void swap(int *a, int *b)
{
	int t;
	t = *a;
	*a = *b;
	*b = t;
}
```
Вызов такой функции будет выглядеть следующим образом:
```c
swap(&x, &y);
```
На время работы функции в ее *локальных* переменных `a` и `b` окажутся *адреса* переменных `x` и `y`. Выражения `*a` и `*b` станут соответственно обозначать переменные `x` и `y`
