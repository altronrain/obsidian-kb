В языке C предусмотрена возможность для определения размеров любых *типов*. Для этого используется *ключевое слово* (*оператор*) `sizeof`. (Это *не функция*)
Так, выражение вида:
```c
sizeof(<тип>)
```
компилятор заменит на число, равное размеру типа. При этом единицей измерения является `char`
Пример:
```c
sizeof(char) --> 1
```

Кроме имен типов можно применять к *именам переменных* и *произвольным выражениям* при этом:
- при применении к типам круглые скобки обязательны
- при применении к именам переменных и выражениям круглые скобки можно не указывать
Пример:
```c
sizeof(short) -> 2 байта
sizeof name -> X байтов
```


При применении к именам типов, можно вывести результат на печать используя спецификатор `%zd`, который появился, начиная со стандарта C99.
Пример:
```c
/* Спецификатор %zd доступен начиная с C99 */

printf("Тип short имеет размер %zd байт(ов).\n", sizeof(short));
printf("Тип int имеет размер %zd байт(ов).\n", sizeof(int));
printf("Тип char имеет размер %zd байт(ов).\n", sizeof(char));
printf("Тип long имеет размер %zd байт(ов).\n", sizeof(long));
printf("Тип long long имеет размер %zd байт(ов).\n", sizeof(long long));
printf("Тип double имеет размер %zd байт(ов).\n", sizeof(double));
printf("Тип long double имеет размер %zd байт(ов).\n", sizeof(long double));
```
Результат работы программы:
```   
Тип short имеет размер 2 байт(ов).  
Тип int имеет размер 4 байт(ов).  
Тип char имеет размер 1 байт(ов).  
Тип long имеет размер 8 байт(ов).  
Тип long long имеет размер 8 байт(ов).  
Тип double имеет размер 8 байт(ов).  
Тип long double имеет размер 16 байт(ов).
```
Как видим, `char` действительно принимается за 1 байт, а осмтальные типы считаются в единицах `char`.

Изучить подробнее размеры типов можно в заголовочных файлах `limits.h` и `float.h`
