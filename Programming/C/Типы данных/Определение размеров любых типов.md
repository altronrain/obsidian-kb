В языке C предусмотрена возможность для определения размеров любых *типов*. Для этого используется ключевое слово `sizeof`.
Так, выражение вида:
```c
sizeof(<тип>)
```
компилятор заменит на число, равное размеру типа. При этом единицей измерения является `char`
Пример:
```c
sizeof(char) --> 1
```

Кроме имен типов можно применять к *именам переменных* и *произвольным выражениям*.
