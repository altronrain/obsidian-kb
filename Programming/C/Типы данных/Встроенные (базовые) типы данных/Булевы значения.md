Определяются ключевым словом `_Bool`. Тип введен стандартом C99.
* для представления истинных значений (`true`)  используется `1`;
* для представления ложных значений (`false`) используется `0`/

Тип `_Bool` -- это целочисленный тип, который требует всего 1 бит памяти, чтобы хранить значения `0` и `1`. Является типом `int` без знака `unsigned`

Фактически, введен для удобства программиста. Ранее и `int` обходились (под капотом в ассемблере это просто числа как были, так и остаются).
