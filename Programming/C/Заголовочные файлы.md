Файлы с суффиксом/расширением `.h` называют *заголовочными файлами*.
В этом файле содержатся только *заголовки* функций, включающий:
* ее имя
* количество принимаемыех параметров
* типы принимаемых параметров
сам *текст функции* в заголовочный файл *не входит*.

Результатом работы компилятора является объектный код, а не машинный.
В объектном файле вместо тела функции будет указание для редактора связей (линкера), что следует откуда-то подставить функцию с указанным именем.
