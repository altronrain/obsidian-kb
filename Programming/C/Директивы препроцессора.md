Вставка полного содержания файла/библиотеки в текст текущей программы выполняется директивой `#include`

При этом, указание имени файла в угловых скобках  означает, что файл следует искать в системных директориях (считается, что так следует включать любой файл, не являющийся сам по себе частью программы)
Пример:
```c
#include <stdio.h>
```
Указание своих собственных файлов производится в двойных кавычках:
```c
#include "mymodule.h"
```
