## Применение оператора
Является вспомогательным оператором для циклов. Может использоваться во всех 3 видах циклов.

Оператор `continue` вызывает *пропуск оставшейся части итерации* и начало новой итерации.

Оператор `continue` воздействует только на структуру (цикл), его содержащую (если циклов несколько, то он будет действовать на тот из них, в который он включен).

Пример:
```c
printf ("Введите результат первой игры (или q для завершения): ");
while (scanf ("%f", &score) == 1) {
	if (score < MIN || score > MAX) {
		printf ("%0.1f -- недопустимое значение. Повторите попытку: ", score);
		continue;
	}
	printf ("Accepting %0.1f:\n", score);
	min = (score < min) ? score : min;
	max = (score > max) ? score : max;
	total += score;
	n++;
	printf ("Введите результат следующей игры (или q для завершения): ");
}
```
Здесь с помощью оператора `continue` исключаем дальнейшую обработку если `score` вне диапазона `0.0 - 100.0` (и выполняется переход к следующей итерации цикла).

Можно было бы переписать цикл и без `continue`, используя условие:
```c
if (score >= 0 && score <= 100) {
	...
}
```
но это дополнительный уровень вложенности на весь цикл, что не всегда удобно.

## Возобновление выполнения цикла
После оператора `continue` где конкретно продолжается выполнение цикла?

1. Для `while` и `do while` следующим после `continue` будет вычисление условия проверки цикла.
2. Для `for` следующим действием будет:
	* вычисление *обновляющего* выражения (3 часть в объявлении `for`)
	* вычисление *проверочного* выражения (2 часть в объявлении `for`)
