Виды циклов в языке C:
* цикл с предусловием
* цикл с постусловием
* цикл, выполняемый известное (вычислимое) количество итераций -- `for`

См. также [[Оператор continue]] и [[Оператор break]]

## Цикл с предусловием (`while`)

Синтаксис:
```c
while ( <условие> ) <оператор>
```
Цикл (оператор) предполагает:
* вычисление условия и выход, если оно ложно
* если условие истинно -- выполнение оператора
* снова вычисление условия (и так пока условие не станет ложным)

## Цикл с предусловием и конечным количеством итераций (`for`)

Он не обязан быть строго арифметическим (он может быть, но не обязан)
Синтаксис:
```c
for ( [<инициализация>] ; [<условие>] ; [<итерация>] ) <оператор>
```
Заголовок цикла `for` состоит из 3 секций, разделенных символом точки с запятой:
* первое из выражений (*инициализация*) вычисляется *перед началом работы* цикла (обычно это присваивание начального значения переменной цикла, но может быть что угодно). Выполняется *один раз*!
* второе выражение (*условие*) представляет собой *условие продолжения* цикла (как в `while`) (пустое -- истина)
* третье выражение (*итерация*) вычисляется каждый раз *после выполнения тела* цикла, то есть перед очередным (но не первым) вычислением условия

Все три секции являются необязательными и могут быть опущены (разделители должны остаться)

В целом по задумке цикл `for` чаще всего нужен для реализации цикла со счетчиком, и здесь в отличие от `while` все будет собрано в объявлении цикла.

Варианты применения:
* без инициализации и без итерации -- ничего не делать
* пропущено условие -- выполнять цикл "бесконечно" (так как считается истиной)
* присутствует только условие -- аналогично циклу `while` с таким условием

Часто для бесконечного цикла используется запись `for(;;)` (*forever*)

![[Pasted image 20260201193354.png]]

> [!NOTE] Цикл `for` может быть не только арифметический
> Не рекомендуется применять цикл `for` в подобном виде, однако ничего не запрещает с точки зрения синтаксиса использовать следующую конструкцию:
> ```
> for(s=0; scanf("%lf", &k) == 1; s += k)
> 	;
> ```
> тело цикла оказалось пустым, ибо все вычисления записаны в условие цикла `for`
> В данном примере подсчитывается сумма прочитынных чисел типа `double`

Цикл `for` является единственным, где точка с запятой `;` ставится не в конце оператора, а в его середине.
Цикл `for` позволяет использовать *больше одной переменной цикла*. Именно для этого (скорее всего) [[Иные (экзотические) операции#Запятая]] была сделана *операцией*
Пример:
```c
for(i=0, j=arr_len -1; i < j; i++, j--) {
	int t;
	t = arr[i];
	arr[i] = arr[j];
	arr[j] = t;
}
```

## Цикл с постусловием (`do while`)

Синтаксис:
```c
do <оператор> while ( <условие> ) ;
```
Условие в цикле `do-while` является условием *продолжения работы* цикла, а не его *завершения*

Цикл (оператор) предполагает:
* выполнение опреатора
* вычисление условия и если оно ложно -- выход из цикла
* если условие истинно -- выполнение цикла с начала

![[Pasted image 20260205220327.png]]

В C *рекомендуется* (но это не обязательно) *записывать тело цикла в виде составного оператора* `{}`

Рекомендуется применять цикл `do-while` только в тех случаях, когда необходимо выполнить хотя бы одну итерацию, например при запросе паролей или иных данных.

Единственная возможность отличить ключевое слово `while` при записи цикла `do-while` от вложенного цикла `while`, это поставить его на одной строке с закрывающей фигурной скобкой от тела
Пример:
```c
do {
	get_event(&event);
	res = handle_event(&event);
} while (res != EV_QUIT_NOW);
```

Оператор `do-while` оканчивается символом точки с запятой `;`. Она входит в его синтаксис

## Операторы управления выполнением цикла
С любым циклом можно выполнить следующие операции:
* досрочное прекращение цикла -- оператор `break;` (именно с `;`, хоть она и избыточна)
* досрочное завершение итерации цикла с продолжением со следующей итерации -- оператор `continue;` (точнее -- игнорирование остатка тела цикла)
	* для циклов `while` и `do-while` выполнение продолжится с вычисления *условия*
	* для цикла `for` выполнение продолжится вычислением выражения *итерация*

Также для преждевременного выхода из цикла (а точнее целиком для возврата из функции) может использоваться оператор `return` (см. [[Возврат из функции]])

## Вложенные циклы
Логично предположить, что одни циклы можно вызывать внутри других. Такие внутренные циклы называются *вложенными*.

Пример:
```c
for (row = 0; row < ROWS; row++) {
	for (ch = 'A'; ch < ('A' + CHARS); ch++)
		printf ("%c", ch);
	printf("\n");
}
```
Пример вывода:
```
$ ./rows1
ABCDEFGHIJ
ABCDEFGHIJ
ABCDEFGHIJ
```
Ход выполнения внутреннего цикла может зависеть от внешнего. Например:
```c
for (row = 0; row < ROWS; row++) {
	for (ch = ('A' + row); ch < ('A' + CHARS); ch++)
		printf ("%c", ch);
	printf("\n");
}
```
Пример вывода:
```
$ ./rows2
ABCDEF
BCDEF
CDEF
DEF
EF
F
```
