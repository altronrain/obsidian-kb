## Оператор ветвления `if else`
Синтаксис оператора ветвления в C:
```c
if (<условие>) <оператор1> [else <оператор2>]
```
Условное выражение обязательно должно быть заключено в круглые скобки `()`

Тип условного выражения может быть:
* целочисленным
* адресным
Значения `0` и нулевой указатель означают "ложь", все остальное -- "истину"

Если результатом вычисления условного выражения стала "истина", выполняется `<оператор1>`, в противном случае -- `<оператор2>`

Синтаксис оператора `if` сам по себе *не предусматривает* точки с запятой, но ее могут потребовать `<оператор1>` или `<оператор2>`

Оператор ветвления `if else` считается *одним простым оператором*, потому не требует вокруг себя фигурных скобок, как для составного блока.

## Оператор `else if`
Служит для упрощения восприятия и написания сложных условий.
Так вместо использования оператора ветвления `if else` в качестве `<оператора2>`
с указанием отступов:
```c
if (kwh <= BREAK1)
	bill = RATE1 * kwh;
else
	if (kwh <= BREAK2)
		bill = BASE1 + (RATE2 * (kwh - BREAK1));
	else
		if (kwh <= BREAK3)
			bill = BASE2 + (RATE3 * (kwh - BREAK2));
		else
			bill = BASE3 + (RATE4 * (kwh - BREAK3));
```
можно написать более понятно и явно обозначить имеющийся выбор:
```c
if (kwh <= BREAK1)
	bill = RATE1 * kwh;
else if (kwh <= BREAK2)
	bill = BASE1 + (RATE2 * (kwh - BREAK1));
else if (kwh <= BREAK3)
	bill = BASE2 + (RATE3 * (kwh - BREAK2));
else
	bill = BASE3 + (RATE4 * (kwh - BREAK3));
```

Компилятору же на отступы все равно, по этому ему без разницы какую форму использовать.

## Образование пар `if` и `else`
Так как компилятор игнорирует отступы, то важно понимать, как он формирует пары `if` и `else` между собой.

Если отсутствуют блочные фигурные скобки `{}` (пусть даже и для простого оператора внутри), то `else` будет отнесен к последнему `if` оператору, стоящему перед ним.
Пример
```c
if (number > 6)
	if (number < 12)
		printf ("Вы закончили игру!\n");
else
	printf ("К сожалению, вы потеряли право хода!\n");
```
не смотря на визуальную группировку `else` к первому оператору `if`, компилятор отнесет его к условию `if (number < 12)`

Таким образом работа фрагмента программы будет давать следующие результаты:
* Число 6 и менее -- нет результата
* Числа от 7 до 12 -- вывод сообщения "Вы закончили игру!"
* Числа от 12 и выше -- вывод сообщения "К сожалению, вы потеряли право хода!" 

Для того чтобы осуществить группировку `else` с первым условием, необходимо внутренний оператор облечь в фигурные скобки:
```c
if (number > 6) {
	if (number < 12)
		printf ("Вы закончили игру!\n");
}
else
	printf ("К сожалению, вы потеряли право хода!\n");
```

Так, работа фрагмента программы будет давать следующие результаты:
* Число 5 и менее --вывод сообщения "К сожалению, вы потеряли право хода!" 
* Числа от 7 до 12 -- вывод сообщения "Вы закончили игру!"
* Числа от 12 и выше -- нет результата.
