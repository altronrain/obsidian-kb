## Основы рекурсии

Способность функции вызывать саму себя называется *рекурсией*.

Хоть сама по себе такая особенность и не является какой-то специализированной конструкцией для управления ходом программы, рекурсия тесно связана с циклами (часто одну и ту же задачу можно решить как 1-2 циклами, так и рекурсией).

Основная задача при организации рекурсии (как, в прочем и при организации цикла): *предусмотреть* проверку условия для *завершения* рекурсии (в данном случае. внутри самой функции).

Плюсы рекурсии:
* удобна и элеганта в применении для определенных задач (где сложно организовать цикл)
* для ряда задач предлагает простейшее решение (в целом равносильно п.1)

Минусы рекурсии:
* сложнее для восприятия чем циклы
* эффективность проигрывает циклам
	* каждый вызов порождает выделение памяти на стеке для новых экземпляров локальных переменных и аргументов (в случае двойной рекурсии рост по экспоненте)
	* каждый вызов сам по себе требует времени, потому рекурсия медленнее
* трудно документировать и сопровождать

Пример организации рекурсии:
```c
void up_and_down (int n)
{
	printf ("Уровень %d: ячейка n %p\n", n, &n); /* ascend */
	if (n < 4) up_and_down (n + 1);
	printf ("УРОВЕНЬ %d: ячейка n %p\n", n, &n); /* descend */

}
```
Вывод программы (включая адреса экземпляров переменной n):

По достижении "пика", в данном случае, функция отрабатывала до конца (2 вызов`printf()` и возвращала управление в предыдущий экземпляр функции `up_and_down()`)

То есть:
* операторы *до* рекурсивного вызова выполняются в том же самом порядке, в каком эти функции вызывались (1-2-3-4...)
* операторы *после* рекурсивного вызова выполняются в порядке, обратном тому, в каком эти функции вызывались

## Хвостовая рекурсия

Простейшая форма рекурсии, когда рекурсивный вызов находится *в конце* функции непосредственно перед оператором `return`. Также может называться *концевой*.
Такая рекурсия подобна одному циклу.

Пример (за исключением последнего случая когда `ans = 1`):
```c
/* вычисление факториала через хвостовую рекурсию: n * (n - 1)! */
long rfact (int n)
{
	long ans;

	/* хвостовая рекурсия для всех вызовов кроме n = 0 */
	ans = (n > 0 ? n * rfact (n - 1) : 1);

	return ans;
}
```
