Составной олператор в языке C представляет собой фигурные скобки `{}` и предназначен для объединения нескольких операторов в один (как в Pascal составным оператором были ключевые слова `begin` и `end`)

В C составной оператор является полноценным *блоком* -- фрагметом программы, для которого *возможны локальные описания* (переменных)

Локальные переменные в ANSI C описываются в самом начале блока, сразу *после* открывающей фигурной скобки и *до* первого оператора
Пример:
```c
if(a[i] > a[i+1]) {
	int t;
	t = a[i];
	a[i] = a[i+1];
	a[i+1] = t;
	flag = 1;
}
```
Переменная `t` не будет видна за пределами условия, ее не будет видно во всей функции.
