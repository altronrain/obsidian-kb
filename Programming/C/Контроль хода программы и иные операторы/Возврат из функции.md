Для возврата управления из функции используется оператор `return`
Он несет двойную нагрузку:
* определяет значение, которое вернет функция
* завершает выполнение функции, в том числе досрочно

Синтаксис:
```c
return <выражение> ;
```
где вычисленное выражение и будет являться возвращаемым значением функции

Для функций, у которых не указан тип возвращаемого значения (ввиду его отсутствия) `void`, в них оператор принимает форму:
```c
return ;
```
При этом `void` функция может завершиться сама, дойдя до конца . Так что подобная форма оператора `return` используется только при необходимости досрочного завершения `void` функции

Для не `void` функций (то есть возвращающих значение) всегда необходим оператор `return` с указанием выражения. Следует внимательно смотреть на предупреждения компилятора на этот счет
