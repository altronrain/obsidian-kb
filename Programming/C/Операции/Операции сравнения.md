Набор операций сравнения:
* `<` меньше
* `>` больше
* `<=` меньше или равно
* `>=` больше или равно
* `==` проверка на равенство (главеное не путать с операцией присваивания `=`)
* `!=` проверка на неравенство

Результат операции сравнения -- целое число:
* `0` ложь
* `1` истина (или любое другое число)

На практике не следует писать выражения со сравнениями и присмаиваниями вида:
```c
x = (a < b) + (a > c) - (c != t);
```
это сильно затрудняет анализ и просмотр кода
