Набор операций сравнения:
* `<` меньше
* `>` больше
* `<=` меньше или равно
* `>=` больше или равно
* `==` проверка на равенство (главеное не путать с операцией присваивания `=`)
* `!=` проверка на неравенство

По приоритету подразделяются на 2 группыы:
* больший приоритет: `< <= > >=`
* меньший приоритет: `== !=`

Результат операции сравнения -- целое число:
* `0` ложь
* `1` истина (или любое другое число, в том числе и отрицательные)

На практике не следует писать выражения со сравнениями и присваиваниями вида:
```c
x = (a < b) + (a > c) - (c != t);
```
это сильно затрудняет анализ и просмотр кода

Операции сравнения выполняются слева направо:
```c
ex != wye == zee
# эквивалент
(ex != wye) == zee
```
