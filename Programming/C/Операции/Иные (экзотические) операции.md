## Операция вызова функции
Операция вызова функции считается (и является) арифметической операцией

*Первым* операндом операции *вызова функции* является ее имя (или ее адрес), а количество операндов (арность) может быть любым, но не меньше одного.
Так, унарная операция вызова функции принимает вид:
```c
f()
```
унарная, так как нет более операндов кроме имени функции

Запись *без скобок* являет собой *адрес функции*, так как именно скобки являются операцией вызова

При вызове функции с параметрами количество операндов равняется количеству параметров +1

## Операция индексирования
Операция индексирования считается (и является) арифметической операцией
Пример:
```c
v[<индекс>]
```

## Тернарная условная операция
Исходя из названия:
* имеет 3 аргумента
* записывается в виде 3 выражений
* после первого выражения ставится `?`
* после второго выражения ставится `:` для отделения от третьего

Логика вычислений:
* вычисляется первое выражение и рассматривается как логическое (истина/ложь)
	* если получилась *истина* -- вычисляется *второе* выражение
	* если получилась *ложь* -- вычисляется *третье* выражение
Примеры:
```c
x = cond(y) ? pro(y) : contra(y);
max = a > b ? a : b;
```
из двух операндов выбирается один, что важно, он и *не вычисляется*

## Запятая
Самая экзотическая операция в языке C
Выражение вида:
```c
a,b
```
Порядок вычислений:
* вычисляется `a`
* вычисляется `b`
* значением всего вместе становится результат вычисления `b`

Тем самым проявляется свойство запятой как *точки следования* (см. [[Побочный эффект и точки следования#Точки следования]]).
*Все* побочные эффекты слева от запятой учитываются до перехода вправо от запятой.

Пример применения запятой как точки следования:
```c
ounces++, cost = ounces * FIRST_OZ;
```
гарантирует инкремент переменной `ounces` и использования ее *нового* значения во втором подвыражении, что важно для вычисляемого значения `cost`

Пример правой части как результата вычисления:
```c
houseprice = 249,500;
/* равносильно: */
houseprice = 249;
500;

houseprice = (249,500);
/* равносильно: */
houseprice = 500;
```
Более "жесткая" демонстрация правой части как результата вычисления:
```c
x = (y = 3, (z = ++y + 2) + 5);
/* Порядок вычислений: */
y = 3;
y = 4;
z = 6;
x = 11;
```

Во *всех* случаях можно обойтись без этой операции, где она используется.

Не всегда символ запятой является операцией, например при перечислении параметров функции они именно синтаксический разделитель

## Операция приведения типов

См. также [[Programming/C/Типы данных/Преобразование типов]]

Помимо автоматического преобразования типов в некоторых ситуациях можно *потребовать* выполнения нужного типа преобразований, либо документировать тот факт, что вы *знаете* о выполнении преобразования типов в конкретном выражении.
Для этого предусмотрена операция *приведения типов*.

Внешний вид операции приведения:
```c
(тип)
```
где вместо слова 'тип' указывается необходимый тип.

Пример:
```c
int mice;
...
mice = 1.6 + 1.7;              /* 3.3 -> после усечения = 3 */
mice = (int) 1.6 + (int) 1.7;  /* 1 + 1 = 2 (усечение до слдожения) */
```
