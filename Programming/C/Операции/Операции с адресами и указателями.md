Операции для работы с адресами и указателями:
* `&` операция взятия адреса (от побитового И отличается количеством операндов, так как унарная)
* `*` операция разыменования (от умножения отличается количеством операндов, так как унарная)
* `.` выборка поля из структуры или объединения
* `->` выборка поля из структуры или объединения по имеющемуся адресу

Взятие адреса (`&`) *от значения* переменной *вернет адрес* ячейки памяти (не физический, а некоего уровня плоской абстракции), в которой содержится значение данной переменной:
```c
printf ("Внутри main() pooh = %d\t&pooh = %p\n", pooh, &pooh);
...
Внутри main() pooh = 2          &pooh = 0x7ffe3dedb78c
```

Разыменование (`*`) обратно операции взятия адреса. Применяется для *определения значения* переменной, на которую *указывает* указатель.
```c
ptr = &bah;
val = *ptr;

/* то же самое вместе, что и */
val = bah;
```

Операции (`.` и `->`) применяются для обращения к полям структур:
```c
item1.next

a->b /* то же самое что (*a).b */
```
