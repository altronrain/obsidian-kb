В C существуют операции *знака* :
* `-`
* `+`
это унарные операции, обычно применяемые к переменным. 

Унарная операция `-` предписывает изменить знак переменной, к которой она применена, на противоположный:
```c
rocky  = -12;
smokey = -rocky; /* smokey -> 12 */
```

Унарная операция `+` введена стандартом С90.  Сама по себе она ничего не делает, лишь разрешает использовать запись вида:
```c
val = +4;
```
что ранее считалось недопустимой конструкцией
