Набор логических операции:
* `&&` конъюнкция (логическое и)
* `||` дизъюнкция (логическое или)
* `!` унарная операция отрицания
* `!=` используется для логического исключающего или, так как это то же самое not equal
* отрицание XOR, соответственно можно выразить как `==` (equal)

В C используется схема *ленивых вычислений*, то есть если значение выражения уже определено, второй операнд не вычисляется.
Примеры:
```c
f() || g() /* g() не вычисляется если f() вернула истину */
f() && g() /* g() не вычисляется если f() вернула ноль (ложь) */
```
