Функции отображения `touppe()` и `tolower()` не *модифицируют* исходное значение, а *возвращают* измененное значение.
```c
tolower(ch)      /* ch остается неизменной */

ch = tolower(ch) /* корректный способ модифицировать ch
                    присвоить ей возвращаемое значение */
```
