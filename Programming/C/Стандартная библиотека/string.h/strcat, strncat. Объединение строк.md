## Функция `strcat()`
Функция `strcat()` выполняет объединение двух строк. В качестве аргументов, соответственно, принимает две строки (указатели на начало строк).
Пример формального прототипа (описания):
```c
char * strcat(char *string1, const char *string2);
```

Что делает данная функция:
* *копию* второй строки присоединяет в конец первой строки
* объединенная версия становится новой первой строкой (изменяется первый аргумент)
* вторая строка не изменяется
* возвращает значение первого аргумента -- адрес первого символа полученной новой строки

Данная функция не проверяет, помещается ли новая полученная строка в отведенный изначальный буфер для строки, переданной в качестве первого аргумента. По этому при использовании данной функции придется, например, заранее проверять длину с использованием `strlen`, что не очень удобно. Область применения крайне мала.

В качестве первого аргумента не получится передать указатель на строку из неизменяемой области памяти (таких как объявленных в `#define` или указателем на литерал).

## Функция `strncat()`
Данная функция решает существенный недостаток `strcat()`, а именно, позволяет избежать переполнения буфера по невнимательности.
Данная функция принимает дополнительный аргумент аргумент, указывающий максимальное количество добавляемых символов.
Пример формального прототипа (описания):
```c
char * strncat(char * string1, const char *string2, unsigned num);
```

Что делает данная функция:
* *копию* второй строки присоединяет в конец первой строки, но не более `num` символов
	* останавливается либо по достижении `\0`
	* либо по достижении указанного лимита
* объединенная версия становится новой первой строкой (изменяется первый аргумент)
	* в буфере должно хватать места для `num + 1` символов (так как с резервированием места под `\0`)
* вторая строка не изменяется
* возвращает значение первого аргумента -- адрес первого символа полученной новой строки

Аналогично `strcat()` качестве первого аргумента не получится передать указатель на строку из неизменяемой области памяти (таких как объявленных в `#define` или указателем на литерал).
