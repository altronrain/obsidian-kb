Функция `strlen()` выполняет подсчет длины переданной строки (массива символов, завершающегося символом `\0`)

Пример формального прототипа (описания):
```c
size_t strlen(const char *string);
```

Подсчет ведется в октетах (байтах), потому "в лоб" подходит только для ASCII-строк.

Что не менее важно -- при подсчете длины не учитывается нулевой символ
```c
int main (void)
{
	char mesg[] = "test";
	printf ("Length of the \"test\" string is: %zu\n", strlen (mesg));
	return 0;
}
```
Вывод программы:
```
$ ./str_len    
Length of the "test" string is: 4
```
По этому при выделении места (в том же массиве) с применением `strlen()` стоит указывать `strlen (str) + 1`
