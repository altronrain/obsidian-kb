## Функция `strcmp()`
Функция `strcmp()` выполняет сравнение двух строковых литералов. В качестве аргументов, соответственно, принимает две строки (указатели на начало строк).
Пример формального прототипа (описания):
```c
int strcmp(const char *string1, const char *string2);
```

Что делает данная функция:
* сравнивает переданные ей строковые литералы
	* что важно, не массивы, а только содержимое до первого встреченного `\0` символа
* возвращает `0` если переданные литералы совпадают
* возвращает ненулевое значение, если литералы не совпадают

Сравнение выполняется с точностью до символа, например, оно регистрозависимо.

Возвращаемое значение, согласно стандарту, следующее:
* `0` -- если строки совпадают полностью
* положительное значение -- если первая строка в последовательности сопоставления (по кодам ASCII символов) следует после второй
* отрицательное значение -- если первая строка в последовательности сопоставления (по кодам ASCII символов) предшествует второй

Стандарт не регламентирует, какие именно значения это должны быть. Одни компиляторы будут возвращать `0`, `1`, `-1`, а иные могут возвращать разницу между ASCII символами несовпадающего элемента.

Пример:
```
strcmp ("A", "A") возвращает 0
strcmp ("A", "B") возвращает -1
strcmp ("B", "A") возвращает 1
strcmp ("C", "A") возвращает 1
strcmp ("Z", "a") возвращает -1
strcmp ("apples", "apple") возвращает 1
```
В последнем случае сравнение выполняется между  символом`\0` и символом `s`, где `s` конечно же следует после `\0`, который является первым ASCII-символом.

Важно отметить, что `sctcmp()` работает со строками, а не с символами. Для сравнения символов вполне хватает операций отношения (это же сравнение целых чисел между собой).

## Функция `strncmp()`
Данная функция сравнивает строки до тех пор, пока не обнаружит в них различия либо пока не сравнит `n` символов в строках, указанное в третьем аргументе.
Пример формального прототипа (описания):
```c
char * strncat(char * string1, const char *string2, size_t num);
```

Что делает данная функция:
* сравнивает переданные ей строковые литералы
	* что важно, не массивы, а только содержимое до первого встреченного `\0` символа
	* либо до нахождения различий
	* либо по достижении `n` символов, указанных третьим аргументом
* возвращает `0` если переданные литералы совпадают
* возвращает ненулевое значение, если литералы не совпадают

То есть, данная функция может сравнить только часть строк (как поиск по подстроке).

Пример:
```c
#include <stdio.h>
#include <string.h>
#define LISTSIZE 6

int main (void)
{
	const char * list[LISTSIZE] =
	{
		"astronomy", "astatism",
		"astrophysics", "ostracism",
		"asterism", "astrolabe"
	};
	int count = 0;

	for (int i = 0; i < LISTSIZE; i++)
		if (strncmp (list[i], "astro", 5) == 0) {
			printf ("Found: %s\n", list[i]);
			count++;
		}

	printf ("%d entries in the list starts with \"astro\"\n", count);

	return 0;
}
```
Результат выполнения программы представлен ниже:
```
$ ./starsrch
Found: astronomy
Found: astrophysics
Found: astrolabe
3 entries in the list starts with "astro"
```
