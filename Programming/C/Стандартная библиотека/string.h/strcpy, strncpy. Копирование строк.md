## Функция `strcpy()`
Функция `strcpy()` выполняет копирование одной строки в другую. В качестве аргументов, соответственно, принимает две строки (*указатели* на начало строк).
Пример формального прототипа (описания):
```c
char * strcpy(char *target, const char *source);
```

Что делает данная функция:
* копирует строковый литерал из второго аргумента (*источника*) в первый аргумент (*цель*)
	* важно, чтобы первый аргумент имел выделенное место под хранение строки (массив с достаточным размером или *проинициализированный* указатель)
	* второй аргумент может быть указателем, именем массива, строковой константой (чем угодно)
* возвращает *адрес первого символа* *полученной* строки

Можно изменять имеющийся массив, передавая в качестве первого аргумента адрес со смещением, тем самым заменяя часть имеющегося литерала.
Функция `strcpy()` также выполняет копирование нулевого символа `\0`, так что иные символы, что могли быть за вставляемой фразой, могут стать недоступными.

Данная функция не проверяет, помещается ли новая полученная строка в отведенный изначальный буфер для строки, переданной в качестве первого аргумента. По этому при использовании данной функции придется, например, заранее проверять длину с использованием `strlen`, что не очень удобно.

Пример использования:
```c
/* copy2.c -- использование функции strcopy() */
#include <stdio.h>
#include <string.h>
#define WORDS "worst"
#define SIZE 40

int main (void)
{
	const char * orig = WORDS;
	char copy[SIZE] = "Be the best you can be.";
	char * ps;

	puts (orig);
	puts (copy);
	ps = strcpy (copy + 7, orig);
	puts (copy);
	puts (ps);

	return 0;
}
/*
$ ./copy2
worst
Be the best you can be.
Be the worst
worst
*/
```

## Функция `strncpy()`
Более безопасный способ копирования строк.
Данная функция копирует строку до тех пор, пока не скопирует всю строку-источник (включая нулевой символ), либо пока не скопирует `n` символов, указанное в третьем аргументе.
Пример формального прототипа (описания):
```c
char * strcpy(char *target, const char *source, size_t num);
```

Что делает данная функция:
* копирует строковый литерал из второго аргумента (*источника*) в первый аргумент (*цель*)
	* важно, чтобы первый аргумент имел выделенное место под хранение строки (массив с достаточным размером или *проинициализированный* указатель)
	* второй аргумент может быть указателем, именем массива, строковой константой (чем угодно)
	* прекращает копирование либо по достижении конца строки-источника (включая `\0`), либо по достижении лимита, указанного третьим аргументом.
* возвращает *адрес первого символа* *полученной* строки

Пример:
```c
/* copy3.c -- использование функции strncpy() */
#include <stdio.h>
#include <string.h>
#define SIZE 40
#define TARGSIZE 7
#define LIM 5

char * s_gets (char * st, int n);

int main (void)
{
	char qwords[LIM][TARGSIZE];
	char temp[SIZE];
	int i = 0;

	printf ("Input %d words, starting wityh 'q' letter:\n", LIM);

	while (i < LIM && s_gets (temp, SIZE)) {
		if (temp[0] != 'q')
			printf ("%s do not starts with 'q'!\n", temp);
		else {
			strncpy (qwords[i], temp, TARGSIZE - 1);
			qwords[i][TARGSIZE - 1] = '\0';
			i++;
		}
	}

	puts ("The list of accepted words:");

	for (i = 0; i < LIM; i++)
		puts (qwords[i]);

	return 0;
}
...
```
