Функция `strol()` из заголовочного файла `stdlib.h` также применяется для преобразования числа из строкового представления (записи) в целочисленное для дальнейшей обработки (`long`). Имеет расширенный функционал в сравнении с `atoi()`

Функция принимает 3 аргумента:
* указатель на строку, которую преобразуем;
* *адрес указателя*, в который будет занесен адрес символа, на котором прекратился ввод
* основание системы счисления

Пример формального прототипа (описания):
```c
long strtol(const char *nptr, char **endptr, int base);
```

Обратить внимание именно на адрес указателя во втором аргументе:
```c
char number[LIM];
char * end;

strtol (number, &end, 10);
```

Принцип работы функции:
```c
"213" (оно же {'2', '1', '3', '\0'})
# будет преобразовано к
213
```

Действие ее подобно тому, как `scanf()` со спецификатором `%d` преобразует введенное alpha-numeric представление числа в целочисленное значение.

Данная функция закончит преобразование на первом же символе, не являющимся alpha-numeric. Занесет его в переданный указатель (по данному указателю сможем вывести остаток строки)
```c
"42text" -> 42
# end будет содержать "text"
```

Преобразование выполняется в системах счисления вплоть до основания 36. Буквы алфавита могут задействуются и корректно преобразуются в зависимости от системы счисления.

Пример:
```c
while (s_gets (number, LIM) && number[0] != '\0') {
	value = strtol (number, &end, 10); /* основание 10 */
	printf ("десятичный ввод, десятичный вывод: %ld, прекращен на %s (%d)\n",
			value, end, *end); /* end - указатель на символ, *end - сам символ */
	value = strtol (number, &end, 16); /* основание 16 */
	printf ("шестнадцатиричный ввод, шестнадцатиричный вывод: %ld, прекращен на %s (%d)\n", value, end, *end);
	puts ("Следующее число:");
}
```
Вывод:
```
$ ./strcnvt
Введите число (или пустую строку для выхода из программы):
10
десятичный ввод, десятичный вывод: 10, прекращен на (0)
шестнадцатиричный ввод, шестнадцатиричный вывод: 16, прекращен на (0)
Следующее число:
10atom
десятичный ввод, десятичный вывод: 10, прекращен на atom (97)
шестнадцатиричный ввод, шестнадцатиричный вывод: 266, прекращен на tom (116)
Следующее число:

Программа завершена.
```
Во втором случае `a` для варианта с 16 системой счисления было обработано корректно и ввод прекратился на символе `t`, имеющем код `116`.

Подобные функции:
* `strtoul()` -- выполняет преобразование к целочисленному без знака (`unsigned long`). Также принимает 3 аргумента;
* `strtod()` -- выполняет преобразование к числу с плавающей точкой (`double`). Работает только в десятичной системе счисления и принимает 2 аргумента.
