Функция `atoi()` из заголовочного файла `stdlib.h` применяется для преобразования числа из строкового представления (записи) в целочисленное для дальнейшей обработки (`int`)

А именно:
```c
"213" (оно же {'2', '1', '3', '\0'})
# будет преобразовано к
213
```

Действие ее подобно тому, как `scanf()` со спецификатором `%d` преобразует введенное alpha-numeric представление числа в целочисленное значение.

Данная функция закончит преобразование на первом же символе, не являющимся alpha-numeric. 
```c
"42text" -> 42
```

Преобразование выполняется в десятичной системе счисления.

Пример:
```c
/* hello.c -- преобразование аргумента командной строки в число с помощью atoi() */
#include <stdio.h>
#include <stdlib.h>

int main (int argc, char **argv)
{
	int i, times;

	if (argc < 2 || (times = atoi (argv[1])) < 1)
		printf ("Использование: %s положительное-число\n", argv[0]);
	else
		for (i = 0; i < times; i++)
			puts ("Хорошего дня!");

	return 0;
}
```
Вывод:
```
$ ./hello
Использование: ./hello положительное-число

$ ./hello 3
Хорошего дня!
Хорошего дня!
Хорошего дня!
```

Также видел ее применение при преобразовании IP-адресов.

Варианты `atol()` и `atof()` выполняют преобразование к `long` и `doudle` типам, соответственно.
