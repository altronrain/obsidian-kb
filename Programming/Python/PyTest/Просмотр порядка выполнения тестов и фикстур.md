Для просмотра порядка выполнения тестов и фикстур, в том числе фаз `SETUP` и `TEARDOWN` используется флаг `--setup-show`.
\
Пример:
```
<имя_теста> 
SETUP    S pytestconfig  
...
SETUP    S browser (fixtures used: launch_browser)  
SETUP    S device (fixtures used: pytestconfig)  
...
SETUP    S browser_context_args (fixtures used: browser_context_args)  
 SETUP    P edge_ip (fixtures used: cfg)  
		...
       SETUP    F context (fixtures used: new_context)  
       SETUP    F page (fixtures used: context)  
       SETUP    F auth_page (fixtures used: edge_ip, page)  
       SETUP    F dashboard_page (fixtures used: edge_ip, page)  
       SETUP    F setup (fixtures used: auth_page, dashboard_page)  
       <тело_теста>
       TEARDOWN F setup  
       TEARDOWN F dashboard_page  
       TEARDOWN F auth_page  
       TEARDOWN F page  
       TEARDOWN F context  
		... 
 TEARDOWN P edge_ip  
TEARDOWN S browser_context_args  
... 
TEARDOWN S device  
TEARDOWN S browser  
...  
TEARDOWN S pytestconfig
```
