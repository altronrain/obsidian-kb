При проверке кода (См. [[Методы белого ящика]] или [[Критерии белого ящика]]) тестами полезно иметь статистическую информацию о том, какой код выполнялся во время проверок, а какой выполнен не был.

Существует два основных метода измерения покрытия:
* линейное покрытие (line coverage). Соотношение выполненных строк кода к общему количеству строк кода
* покрытие "веток"/решений (branch coverage). 

Для оценки покрытия кода программы существуют специализированные сторонние [[Модули PyTest]], например `pytest-cov` или `coverage.py`.
