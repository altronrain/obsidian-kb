Является антипаттерном. Так делать не нужно никогда.

Правильный путь -- делать копию списка, который собираемся модифицировать, и итерировать ее в цикле, выполняя действия с оригинальным списком (или наоборот)

Самый простой вариант -- явная копия в объявлении цикла:
```python
for item in list(some_list)
	...
```
Функция `list()` принимает итерируемые объекты и преобразует их в список. 
Ничто не мешает передать туда список и получить новый список.
Подтверждение:
```python
a = ['a', 'b', 'c']
b = list(a)

print(f'{a=}')
print(f'{b=}')
print(f'{id(a)}')
print(f'{id(b)}')
```
Результат:
```python
a=['a', 'b', 'c']  
b=['a', 'b', 'c']  
139994031938176  
139994043548736
```
