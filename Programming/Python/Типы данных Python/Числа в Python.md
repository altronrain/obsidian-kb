Рассмотрим представление чисел в Python.
Существует несколько подтипов:
* `integer` -- целые числа
* `float` -- числа с плавающей точкой
* `complex` -- комплексные числа
## Integer. Целые числа
Конструктор: `int()`
Пример литералов типа `integer`
```python
7     2147483647                        0o177    0b100110111
3     79228162514264337593543950336     0o377    0xdeadbeef
      100_000_000_000                   0b_1110_0101
```

При использовании `int()` можно конвертировать:
* *числа* к десятичной системе счисления
* *числа* с плавающей точкой к целым числам с округлением
* строки, байты, массивы байтов, которые представляют собой числа в какой-то системе счисления (в том числе с учетом знака)

Пример конвертации строкового литерала к целому числу:
```python
In [1]: int('128')  
Out[1]: 128
```
Пример конвертации строкового литерала к целому числу по указанному основанию
```python
In [2]: int('10', base=2)  
Out[2]: 2
```
При указании основания `0` Python попытается вычислить основание из указанного строкового литерала:
```python
In [3]: int('0b100', base=0)  
Out[3]: 4
```

Говоря о преобразовании к `integer` нельзя не отметить функции `bin()`, `oct()` и `hex()`, выполняющих преобразование из числа в строковый литерал, являющийся представлением данного числа в указанной системе счисления:
```python
In [14]: bin(42)  
Out[14]: '0b101010'

In [15]: oct(69)  
Out[15]: '0o105'

In [18]: hex(16873)  
Out[18]: '0x41e9'
```

## Float. Числа с плавающей точкой
Конструктор: `float()`
Пример литералов типа `float`
```python
3.14    10.    .001    1e100    3.14e-10    0e0    3.14_15_93
```

При использовании `float()` он конвертирует *строку* или *число* в число с плавающей точкой, если это возможно.

Примеры конвертации:
```python
In [20]: float('122.15')  
Out[20]: 122.15  
  
In [21]: float(512)  
Out[21]: 512.0
```

Про точность базового типа `float()` смотри [[Машинное представление чисел с плавающей точкой]]
## Complex. Комплексные числа
Конструктор: `complex()`
Пример литералов типа `complex`
```python
3.14j   10.j    10j     .001j   1e100j   3.14e-10j   3.14_15_93j   (3+4j)
```

При использовании `complex()` можно указать вещественную и мнимую части
```python
In [27]: complex(1)  
Out[27]: (1+0j)  
  
In [28]: complex(2, 3)  
Out[28]: (2+3j)
```
Создание комплексных чисел без использования `complex()`
```python
In [29]: 5 + 2j  
Out[29]: (5+2j)
```
