В общем случае генерация *псевдослучайных* чисел выполняется на основе некоего *random seed* (не путать с генерацией случайных чисел)

Потенциально, использование random seed позволяет повторить последовательность, если передать то же самое начальное значение, что может быть полезно при отладке.

Для генерации таких псевдослучайных последовательностей в Pascal предусмотрены:
* процедура `randomize`
* функция `random`

Процедура `randomize` вызывается ровно один раз (и заносит метку времени в переменную `randseed`, с которой напрямую не работаем)

Функция `random` вызывается либо без аргументов (тогда возвращает `real` в диапазоне \[0, 1)), либо с передачей `longint`, тогда вернет целое в диапазоне \[0, `longint`)
Также эта функция обладает *побочным эффектом*, так как меняет переменную `randseed`, чтобы при следующем вызове вернуть другое число.
