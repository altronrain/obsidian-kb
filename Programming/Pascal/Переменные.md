Pascal -- язык со строгой типизацией. Тип переменной задается в момент ее объявления и не может быть изменен в дальнейшем.
Тип переменной фиксирует множество значений, их машинное представление и набор операций, определенных над этими значениями.

С переменными в определенный момент связано некое значение (переменная *хранит* значение, значение *содержится* в переменной).

Переменные объявляются в секции `var`
Пример:
```pascal
var
	x, y, z: integer
```

*Инициализация*  -- указание значения переменной непосредственно при объявлении.
Пример:
```pascal
var
	x: integer = 25;
```
В противном случае, переменная является неинициализированной (и фактически содержит мусор).
Перед использованием такой переменной необходимо *присвоить* ей значение оператором присваивания `:=`
Пример:
```pascal
x := 79
```
*Присваивание изменяет* начальное (и любое предыдущее) *значение переменной*.

При присваиваниях подобного рода:
```pascal
x := x + 5
```
сначала вычисляется выражение справа (т.е. используется старое значение переменной `x`, затем результат заносится в переменную `x`)
