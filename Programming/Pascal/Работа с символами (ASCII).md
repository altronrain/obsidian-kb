## Запись символов в различных нотациях
Pascal позволяет указывать символы ASCII в следующих нотациях:
* сам символ, заключенный в апострофы (если требуется указать сам апостроф, то будет удвоение '') -- `'a'`,`'g'`,`'Z'`,`'8`
* с помощью кода в десятичной системе через решетку -- `#10`(LF),`#55`(7)
* в "кареточной нотации" (для символов от 1 до 26) -- `^Z`(&), `^J`(LF)
* запись через оператор `chr()` (аналогично записи через решетку)

Так, вместо `writeln` можно написать:
```pascal
write('Hello, world!'#10)
```

При записи через `chr()`, в отличие от других типов записи мы можем заранее и не знать, какой код символа будет передан оператору, там может быть выражение, что позволяет использовать эту запись в теле подпрограмм

## Операторы преобразования
Существует два оператора, позволяющих выполнять преобразование символа к его коду и обратное действие
* `chr(num)` -- преобразует код символа в сам символ, в том числе управляющий
* `ord(symbol)` -- преобразует символ в его код символа

Примеры:
```pascal
chr(55) -> #55 -> '7'
ord('7') -> #55
```

## Операции сравнения над числами
Так как последовательность символов ASCII непрерывна, это позволяет использовать над символами операции сравнения. При этом сравниваются их коды.

Эта возможность позволяет формулировать условия вида:
* "Является ли переданный символ строчной буквой английского алфавита"
* "Является ли переданный символ числом"
```pascal
(c >= 'a') and (c <= 'z')
(c >= '0') and (c <= '9')
```
