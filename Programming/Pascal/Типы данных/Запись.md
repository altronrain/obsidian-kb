Тема тесно связана с [[Пользовательские типы]]
Сложные перменные  отличаются от простых тем, что сами *состоят из переменных* более простых типов.
Сложные переменные бывают двух типов:
* массивы
* записи

Рассмотрим записи

*Записью* называется сложная переменная, состоящая из *нескольких переменных*, называемых *полями*, *типы* которых *могут быть различны*

Поля записи различаются *именами*, собственно по ним *ведется обращение* к элементам.

При объявлении используется ключевое слово `record`
Пример явного указания типа запись:
```pascal
type
	Book = record
		BookName, author, genre: string;
		pages: integer;
		price: real;
	end;
```
После чего можно переменной присвоить созданный тип:
```pascal
var
	book1: Book;
```
Переменная `book1` будет занимать столько памяти, сколько необходимо для размещения всех ее полей в памяти. 

 Над самими записями, как единым целым определено только *присваивание*. Остальные действия выполняются над элементами записей.

Обращение к полям записи осуществляется через точку
```pascal
book.name := 'Wizard of Oz';
book.price := 900.99;
```
Каждое поле записи -- полноценная переменная (и может быть использована как параметр-переменная в функции или процедуре)
