Рассмотрим представление чисел в Pascal.

Существует несколько подтипов:
* `shortint`, `integer`, `longint`, `int64` -- целые *знаковые* числа (соответственно 1-, 2-, 4- и 8-ми байтные)
* `byte`, `word`, `longword`, `qword` -- целые *беззнаковые* числа (соответственно 1-, 2-, 4- и 8-ми байтные)
* `real` -- числа с плавающей точкой (на самом деле там 3 типа, а `real` = `double`):
	* `single`
	* `double`
	* `extended`

Над числами допустимы следующие операции:
* `+` и `-` -- сложение и вычитание
* `*` и `/` -- умножение и деление
* сравнение между собой

Результатом сравнения будет являться булево значение
Тип данных в результате выполнения `+`, `-`, и `*`  над числами *не изменяется*
Тип данных в результате выполнения `/` всегда `real`

Примеры записи целых чисел
```pascal
2     90     $1A7     $23b5     &756    %1000101
```
где запись через:
* `$` -- шестнадцатиричные числа
* `&` -- восьмиричные числа (только FreePascal)
* `%` -- двоичные числа (только FreePascal)

Пример записи чисел с плавающей точкой
```pascal
37.0    2.11    7e3    2.5e-5
```
при записи через научную нотацию указывается:
* мантисса (число от 1 (включительно) до 10)
* через `e` указывается степень 10, на которую необходимо умножить мантиссу
