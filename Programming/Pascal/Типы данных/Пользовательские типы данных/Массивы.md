Тема тесно связана с [[Пользовательские типы]]
Сложные перменные  отличаются от простых тем, что сами *состоят из переменных* более простых типов.
Сложные переменные бывают двух типов:
* массивы
* записи

Рассмотрим массивы

*Массивом* называется сложная переменная, состоящая из *нескольких переменных одного и того же типа*, называемых *элементами массива*.

Обращения к значениям массива идет по *индексам*, представляющими собой *порядковый тип* (чаще всего целые числа, но может быть любой, в том числе `boolean` или `char` или любой диапазон и перечисление, см. [[Пользовательские типы#Порядковый тип]])
В том числе можно подставить туда вычисляемое выражение вместо константы (что и используется в программах)
Пример:
```pascal
a[3]
a[5*i - 8]
```

У переменной массива может быть явно указан тип, либо он может быть объявлен анонимно. При объявлении используется ключевое слово `array`
Пример явного указания типа:
```pascal
type
	real100 = array [1..100] of real;
var
	a, b: real100;
```
Пример анонимного указания типа:
```pascal
var
	a, b: array [1..100] of real;
```

Над самими массивами, как единым целым определено только *присваивание*.
При этом производится копирование содержимого одного массива в другой (что может занять много времени). При этом существует нюанс с анонимным объявлением массива. Компилятор каждый раз создает *"новый тип"*, в связи с чем два подобных объявления:
```pascal
var
	a, b: array [1..100] of real;
	c: array [1..100] of real;
```
сделают невозможным присвоение `a := c`, так как тип для `a` и `b` будет отличаться от типа для `c`. Решается только явным объявлением.

Остальные действия выполняются над элементами массивов.

Как уже говорилось, в качестве индексов может использоваться только порядковый тип. Он може может быть объявлен не анонимно
```pascal
type
	from1to100 = 1..100;
	real100 = array[from1to100] of real;
```

Переменные в массивах можно инициализировать при объявлении:
```pascal
type
	arr5 = array[1..5] of integer;
var
	a: arr5 = (25, 36, 49, 64, 81);
```

Каждый элемент массива -- полноценная переменная (и может быть использована как параметр-переменная в функции или процедуре)

Возможно и комбинирование сложных типов, например массив записей (см. [[Запись]]), запись массивов, запись записей, массив массивов. 
Пример массива записей (что соответствует *таблице*):
```pascal
type
	Book = record
		BookName, author, genre: string;
		pages: integer;
		price: real;
	end;
	ListOfBooks = array [1..100] of Book;
var
	BookStoreList: ListOfBooks;
```
С подобным обращением к элементам:
```pascal
BookStoreList[42].BookName
```

Массив массивов (или *многомерный массив*) имеет свой синтаксис записи как упрощенный вариант. Следующие две записи эквивалентны:
```pascal
type
	array1 = array [1..5, 1..7] of integer;
	array2 = array [1..5] of array [1..7] of integer;
```
