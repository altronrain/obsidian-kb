Константа -- выражение, значение которого всегда одно и то же.
Константа времени компиляции -- константы, значение которых компилятор определяет во время обработки программы

Примеры констант времени компиляции:
* литералы (например `37.0` всегда будет типом `real` с таким значением)
* арифметические выражения без обращения к переменным и функциям (например `6*7` --> компилятор вычислит результат и далее будет работать с ним)

Механзм *именованных констант* позволяет связать с константой времени компиляции некое имя, *идентификатор*.
Это действие выполняется в *секции описания констант* `const`, которую можно располагать в любом месте *между заголовком* и началом *главной части программы*.

Каждая константа состоит из идентификатора, знака равенства, значения, точки с запятой
```pascal
const
	const1 = <value1>;
	const2 = <value2>;
```
