## Взаимная рекурсия
Рекурсия может быть *взаимной*, то есть в простейшем случае одна подпрограмма вызывает другую, а вторая -- снова первую

Для реализации подобного в Pascal необходимо прибегать к *предварительному объявлению (декларации)* подпрограмм, иначе компилятор выдаст ошибку, так как на время обработки одной подпрограммы он встретит имя/вызов другой подпрограммы, о которой он еще ничего не знает. Потому необходимо *заранее объявить подпрограмму

При декларации подпрограммы указываются:
* имя подпрограммы
* параметры вызова
* для функции -- тип возвращаемого значения
* ключевое слово `forward` c точкой с запятой

Пример декларации процедуры и функции:
```pascal
procedure TraverseTree(var p: NodePtr); forward;
function CountValues(p: NodePtr; lim: longint): integer; forward;
```

Таким образом порядок действий при взаимной рекурсии такой:
* декларируется первая из подпрограмм
* описывается (полностью, с телом) вторая из подпрограмм
* описывается (полностью, с  телом) первая из подпрограмм

Если задекларированная подпрограммма не будет описана компилятор выдаст ошибку
