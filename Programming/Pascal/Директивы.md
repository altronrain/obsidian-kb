Директивы, если угодно, некие управляющие последовательности, изменяющие поведение компилятора

## Управление обработкой ошибок
Указание компилятору, что мы будем сами обрабатывать ошибки, выполняется последовательностью `{$I-}`, (включается, соответственно `{$I+}`).
По умолчанию включено.

Пример использования совместно со встроенной функцией `IOResult`, возвращающей код очередной операции ввода:
```pascal
{$I-}
read(x, y);
if IOResult = 0 then
	writeln(x * y)
else
	writeln('I couldn''t pasrse your input')
```

## Изменение операции взятия адреса
По умолчанию результатом операции взятия адреса (см. [[Адреса и указатели#Операция взятия адреса]]) является *нетипизированный адрес*.
Поведение можно отключить директивой `{$T+}`
