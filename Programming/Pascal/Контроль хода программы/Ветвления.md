Ветвление являет собой некоторое *условие*, которое может быть истинным или ложным. В зависимости от результата проверки условия выполняется тот или иной блок кода (команды)

Ветвление может быть полным (предусмотрены операторы для обоих исходов) и неполным (предусмотрены операторы только для истинного результата)

Конструкция оператора ветвления (условного оператора) `if` в Pascal
```pascal
if <условие> then
	<оператор1>
else
	<оператор2>
```
Вся конструкция представляет собой *один* оператор. Но сложный, так как включает внутрь себя иные операторы.

Если же внутри оператора ветвления требуется указать множество операторов, то они обрамляются конструкцией `begin` - `end`. Все что заключено в такую конктрукцию будет являть собой *один сложный оператор*.
Примеры
```pascal
if <условие> then
begin
	<оператор1>;
	<оператор2>
end
```
или допустим следующий вариант записи:
```pascal
if <условие> then begin
	<оператор1>;
	<оператор2>
end
```

Часть конструкции с *else* обязательной не является

Условие представляет собой *логическое выражение*, результатом вычисления которого будет логическое значение (истина или ложь).
