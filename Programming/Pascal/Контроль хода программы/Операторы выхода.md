Рассмотрим операторы выхода в Pascal (см. [[Конструирование программ#Исключения из правил операторы выхода]]).
* `exit` -- досрочное завершение подпрограммы
* `halt` -- досрочное завершение всей программы
* `break` -- досрочное завершение цикла
* `continue` -- досрочное завершение итерации цикла

## exit
Досрочный выход из подпрограммы. Хорошо находит применение при обработке специальных случаев в функциях и процедурах чтобы не городить вложенные `if` структуры, где "главный код" функции будет запрятан а глубине
Пример:
```pascal
ok := false;
if a = 0 then
	exit;
d := b*b - 4*a*c;
if d < 0 then
	exit;
d := sqrt(d);
ok := true
```

## halt
Досрочное завершение всей программы. Может быть применен в любом месте программы, в том числе и в подпрограммах.
При применении без параметров имеет код завершения процесса 0, то есть рапортует операционной системе об успешном завершении.
Может принимать параметр в виде числа от 0 до 255, соответственно такой код завершения будет передавать в ОС при завершении работы. Ненулевой код говорит об ошибке в работе программы.

## continue
Досрочное завершение выполнения *одной итерации цикла*
