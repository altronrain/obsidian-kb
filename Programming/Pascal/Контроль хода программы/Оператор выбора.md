Представляет собой некое обобщение оператора ветвления `if`. В отличие от последнего может иметь сколько угодно вариантов-ветвей.

Конструкция оператора выбора `case` в Pascal
```pascal
case expression_or_var of
	'val1':
		{...}
	'val2', 'val3':
		{...}
	'val4'..'valN':
		{...}
	else
		{...}
```
где:
* в ключевые слова `case` `of` заключается выражение (может быть составным, мсожет иметь любой порядковый тип)
* в качестве ветвлений указаны значения -- результаты вычисления выражения (должны задаваться константами времени компиляции):
	* одиночные
	* несколько через запятую
	* диапазоны значений
* имеется возможность указать ветку `else`, которая будет выполняться если результат вычисления выражения не удовлетворяет предусмотренным значениям

С оператором `case` предпочтительнее использовать [[Пользовательские типы#Перечисляемые типы]] для осмысленных именований веток

Вложенные друг в друга операторы выбора на практике программирования недопустимы. За такое бьют по рукам
