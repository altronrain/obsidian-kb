Программе на Pascal при запуске из командной строки, как и любой программе в Unix-подобных ОС можно *передать* различные *аргументы*.

Для работы с ними в Pascal имеются две встроенные функции:
* `ParamCount` 
* `ParamStr`

Функция `ParamCount` работает без параметров и возвращает целое число -- возвращающее *количество параметров программы без учета имени программы*.

Функция `ParamStr` принимает целое число и возвращает строку (`string`), соответствующую параметру командной строки с указанным номером. 
При передаче `0` вернет имя самой программы.

Пример:
```
./cmdline-parse one two three

ParamCount -> 3
ParamStr(2) -> 'two'
```
