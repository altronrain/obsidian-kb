Кратко рассмотрим различные правила комбинаторики

## Правило суммы
Если два действия *А* и *В* взаимно исключают друг друга, причем действие *А* можно выполнить *m* способами, а *В* – *n* способами, то выполнить одно любое из этих действий (либо *А*, либо *В*) можно *n + m*  способами.

## Количество подмножеств в множестве
Формулу лучше не запоминать, а выводить.
В определенном конечном множестве из *n* элементов, при определении количества подмножеств можно заметить, что каждый элемент может либо входить, либо не входить в подможество. Итого *2* варианта на каждый элемент. То есть получим умножение двоек по числу элементов. По этому общее число подмножеств будет равно:

2^n

Если брать чуть более сложные условия, не конкретизируясь на множествах и подмножествах и двойках, то можно перейти к правилу умножения

## Перестановки
Решают класс задач о размещении *n* элементов на *n* позиций (порядок элементов важен).
### Перестановки без повторений
Применяется, когда среди рассматриваемых *n* элементов *отсутствуют одинаковые*.
![[Pasted image 20241029205852.png]]
### Перестановки с повторениями
Задача все еще расставить *n* элементов на *n* позиций, однако среди рассматриваемых элементов имеется *k* различных типов (*k < n*), то есть *имеются одинаковые*.
![[Pasted image 20241029210246.png]]
В знаменатель помещаются количества экземпляров одинаковых элементов. Общая их сумма должна давать *n*:

*n1 + n2 + n3 + ... + nk = n*

Пример:
Количество перестановок букв слова "Миссисипи"
По типам элементов получается следующее наполнение:
```
1 -- "м"; 4 -- "и"; 3 -- "с"; 1 -- "п"
```

## Размещения
Решают класс задач о размещении *m* из *n* элементов на *m* позиций (порядок элементов важен).
### Размещения без повторений
Применяется, когда среди рассматриваемых *n* элементов *отсутствуют одинаковые*.
![[Pasted image 20241029211610.png]]
Иногда по формулировке задачи может потребоваться сложение размещений по различным основаниям. 
Пример:
Сколько существует способов посторить башню высотой от 1 до 4 кубиков из 4 разноцветных кубиков.
Решение:
* для 4 кубиков -- 4!
* для 3 кубиков -- 4!/1!
* для 2 кубиков -- 4!/2!
* для 1 кубика -- 4!/3!
Итого: 24 + 24 + 12 +4 = 64

### Размещения с повторениями
Также размещаем *m* из *n* элементов на *m* позиций, но среди множества *n* есть одинаковые.
![[Pasted image 20241029212712.png]]
Перекликается с [[Элементы комбинаторики#Правило умножения]]
Так, требуется заполнить *m* позиций, каждая из которых может быть в *n* положений

## Сочетания
Решают класс задач выбора *m* из *n* различных предметов (порядок элементов не имеет значения).
### Сочетания без повторений
Так как мы уже знаем формулу перестановок и формулу размещений без повторений, то не составит труда вывести формулу сочетаний. 
В общем случае, из всего множества вариантов *размещений* нужно удалить варианты, отличающиеся только *перестановками* элементов, так как они более не важны для нас. Любая интересующая нас комбинация из общего списка размещений из *n* по *m* элементов будет представлена в списке столько раз, сколько количество перестановок из *m* элементов. Значит, на это число нужно разделить итоговое число размещений.
![[Pasted image 20241029214128.png]]
### Сочетания с повторениями
Для справки (вывода формулы нет). Выбираем *m* из *r* предметов *n* типов.
![[Pasted image 20241029214519.png]]
Пример задачи:
Покупка 7 пирожных из представленных 4 сортов.
